## Yeoman, generator-angular
**related commands**

```

```

* angular (aka angular:app)
* angular:controller
* angular:directive
* angular:filter
* angular:route
* angular:service
* angular:provider
* angular:factory
* angular:value
* angular:constant

* angular:view
